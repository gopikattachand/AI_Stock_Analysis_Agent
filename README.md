# AI Stock Analysis Agent

An AI-powered stock analysis dashboard built using Python and Streamlit.

## Features

- Live stock market data
- Stock price charts
- Moving averages
- RSI technical indicator
- News sentiment analysis
- AI-generated stock summary
- Buy / Hold / Avoid recommendation engine

## Tech Stack

- Python
- Streamlit
- yfinance
- Pandas
- Plotly
- OpenAI API

## Run Project

```bash
streamlit run app.py
```
